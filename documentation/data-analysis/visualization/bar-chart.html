Bar Chart

Bar chart (also called bar graph) is a graphical display of data using bars of different heights or different lengths. The bars can be plotted vertically or horizontally. A bar chart shows comparisons among discrete categories. One axis of the chart shows the specific categories being compared, and the other axis represents a measured value.

Form

  • Variable(s)num

    Select numerical variables and determine whether you want to use sum, frequency, or mean of of those variables to represent the bar length. Statistiku will generate one bar chart for each variables you select.

  • Group byordered

    Select one or two categorical variables which divide your data into bars.

  • Display

    Customize how the bar chart should be generated. Choose how the bars will be plotted, vertically (default) or horizontally. And if you select 2 groups, you can have stacking bars setting here.

Example

You can use schooldays dataset to compare absent between groups (race, sex, school, or learner).
